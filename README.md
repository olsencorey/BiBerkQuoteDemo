# BiBerk Quote Demo – Workers Comp & BOP

BiBerkQuoteDemo is a full-stack insurance quoting application that mirrors a biBERK-style, cloud-first, API-driven platform for small-business commercial lines.

Backend: ASP.NET Core Web API (.NET 8) with in-memory rating logic for Workers' Compensation (payroll, class codes, state mods, experience mods) and Business Owner's Policy (industry, revenue, property values, location risk).
Frontend: Angular application using standalone components and reactive forms to collect inputs, call the API via HttpClient, and render instant quote results.

Tech Stack: .NET 8 · ASP.NET Core Web API · C# · Angular · TypeScript · Swagger/OpenAPI · CORS
